# 1️⃣ 개념 정리 

## 01. 데이터 분석이란

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
데이터 과학은 통계학, 데이터 분석, 머신러닝, 데이터 마이닝 등을 아우르는 큰 개념이다. 이 중 데이터 분석은 좁은 의미에서 기술통계, 탐색적 데이터 분석, 가설 검정이라는 세 가지로 나눌 수 있다. 이러한 데이터 분석을 수행하기 위한 프로그래밍 언어로는 대표적으로 python과 R이 있다. 책에서는 그 중에서도 python을 다룬다. 파이썬을 통해 분석을 진행하기 위해 필요한 파이썬 패키지들이 있는데, 넘파이, 판다스, 맷플롯립 등이 있다. 넘파이는 배열을 효율적으로 다룰 수 있는 다루기 위한 패키지이며, 판다스는 엑셀 시트처럼 숫자와 문자를 섞어서 표 형태로 저장할 수 있는 데이터프레임을 활용하는 라이브러리이다. 맷플롯립은 파이썬 데이터 시각화를 할 때 사용되며, 사이파이는 수학과 과학 계산을 전문으로 수행하는 패키지이다. 마지막으로 사이킷런은 머신러닝을 수행할 때 주로 사용된다.

## 02. 구글 코랩과 주피터 노트북

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
이 책은 구글 코랩을 이용해서 실습을 진행하는데, 코랩은 구글이 대화식 프로그래밍 환경인 주피터 노트북을 커스터마이징한 것으로, 웹 상에서 코드를 작성할 수 있다. 별도의 설치가 필요 없기 때문에 사용하기 편리한 도구이다. 작성한 코랩 파일은 '노트북'이라고도 불린다. 

노트북에서 '셀'이란 코랩에서 실행할 수 있는 최소 단위로, 코드 또는 텍스트의 덩어리를 의미한다. 노트북은 보통 여러 개의 코드 셀과 텍스트 셀로 이루어진다. 텍스트 셀은 HTML과 마크다운을 혼용해서 사용할 수 있다. 코드 셀은 파이썬 코드를 입력하고 실행할 수 있는 셀이다. 

코랩을 활용하다가 일정 시간 동안 아무 작업을 하지 않으면 런타임이 끊긴다. 런타임이 끊어지면 노트북에서 작성한 모든 것이 사라질 수 있으므로 반드시 다시 연결 버튼을 누른 후 노트북의 처음부터 모든 코드 셀을 다시 실행해야 한다. 노트북 메뉴바의 오른쪽에 있는 RAM, 디스크 아이콘에 마우스 커서를 올리면 연결된 런타임에 대한 자세한 정보를 확인할 수 있다. 코랩 노트북으로 동시에 사용할 수 있는 클라우드 가상 서버는 최대 5개이다. 노트북이 더 필요하다면 실행 중인 노트북을 저장하고 런타임과 연결을 끊어야 한다.

## 03. 이 도서가 얼마나 인기가 좋을까요?

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
이 책에서 소개는 공개 데이터를 볼 수 있는 사이트는 공공데이터포털과 도서관 정보 나루가 있다. 특히 도서관 정보 나루 웹사이트는 도서관별로 대출 데이터를 확인할 수 있다. 대출 데이터를 다운 받으면 CSV 파일이 다운받아지는데, CSV는 콤마로 구분된 텍스트 파일이다.

판다스 시리즈 - 1차원 배열과 비슷
판다스 데이터프레임 - 데이터를 가로 세로로 나열한 2차원 배열과 비슷

판다스에서 CSV 파일을 데이터프레임으로 읽기 : read_csv() 사용
판다스에서 데이터를 제대로 가져왔는지 확인하기 : head() 사용
판다스의 데이터프레임을 CSV 파일로 저장하기 : to_csv() 사용
